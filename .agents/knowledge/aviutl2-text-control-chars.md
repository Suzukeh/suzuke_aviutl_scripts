# AviUtl2 テキスト制御文字リファレンス

出典: https://docs.aviutl2.jp/usage#テキスト

テキストオブジェクトや `obj.load("text", text)` で制御文字が使える。
`obj.load("text", ...)` では色・サイズ・フォントの制御文字が使用可能。

## 色 ` <#[RGB][,RGB]]>`

文字色・影縁色を変更。16進数またはプリセット色名。

```
<#ffffff>            文字色を白
<#000000,ffffff>     文字色=黒, 影縁=白
<#>                  元に戻す
<#red>               プリセット色 "red"
```

## サイズ・フォント `<s[サイズ][,フォント名][,装飾][,縁取りサイズ]>`

```
<s32>                サイズ32
<s+10>               +10 (相対)
<s*1.5>              1.5倍 (相対)
<s>                  元に戻す
<s32,メイリオ>       サイズ32 + メイリオ
```

※フォント変更は `<@>` での変更が推奨

## フォント・装飾 `<@[フォント名][,文字装飾]>`

文字装飾: 0=標準 / 1=影 / 2=影(薄) / 3=縁取り / 4=縁取り(細) / 5=縁取り(太) / 6=縁取り(角)
スタイル: B=太字 / I=斜体 / S=取り消し線

```
<@メイリオ>          メイリオ
<@メイリオ,3>        メイリオ + 縁取り
<@メイリオ,6BI>      メイリオ + 縁取り(角) + 太字 + 斜体
<@>                  元に戻す
```

## フォントスタイル追加/削除 `<@+[スタイル]>` / `<@-[スタイル]>`

```
<@+B>                太字追加
<@-B>                太字削除
```

## プリセット `<$[プリセット名]>`

テキストプリセットを適用。`<$>` で初期値に戻す。

## 表示速度 `<r[速度]>`

```
<r10>                速度10
<r>                  元に戻す
```

## 表示ウェイト `<w[待機秒]>`

`*` 付きだと表示文字数を掛ける。

```
<w5>                 5秒待機
<w*0.2>              表示文字数×0.2秒
```

## 表示クリア `<c[待機秒]>`

指定秒後に文字をクリア。未指定=即時。

```
<c>                  即時クリア
<c5>                 5秒後にクリア
<c*0.2>              表示文字数×0.2秒後にクリア
```

## 座標 `<p[X][,Y][,Z]>`

```
<p+10>               X+10 (相対)
<p20,40,80>          絶対指定
<p+20,+40>           相対
<p>                  基準座標に戻す
```

## 字間・行間 `<gw[字間]>` / `<gh[行間]>`

```
<gw10> <gh10> <gw> <gh>
```

## 文字変形 `<tw[横]>` / `<th[幅]>` / `<tr[角度]>`

```
<tw0.8>              横スケール0.8
<th0.8>              幅スケール0.8
<tr90>               90度回転
<tr+45>              +45度 (相対)
<tr>                 元に戻す
```

## 絵文字 `<&[名前]>`

```
<&絵文字> <&いいね>
```

## ふりがな(ルビ) `</>[漢字]<![スケール][+] >[ふりがな]></>`

```
</>制御文字<!>せいぎょもじ</>
</>行間拡張<!+>ぎょうかんかくちょう</>
</>字間<!0.4><gw8><#ff8888>じかんちょうせい<#><gw></>
```

## コメント `<//[コメント]//>`

```
<// あとで修正する //>   改行含めてコメント可
</>                     文字列ブロック区切り
```

## スクリプト `<?[スクリプト]?>` / `<?=[mes()の引数]?>`

```
<?obj.rz=obj.time*360?>
<?=string.format("%02d:%02d",obj.time/60,obj.time%60)?>
```

## スクリプトから使う場合の注意

- `obj.load("text", text)` では **色・サイズ・フォント** の制御文字が使用可能
- **回転** は制御文字 `<tr角度>` で可能
- `obj.setfont()` で事前設定したフォントは `obj.load("text")` に適用される
- ※テキストオブジェクトには `obj.load("text",...)` は使用出来ない
