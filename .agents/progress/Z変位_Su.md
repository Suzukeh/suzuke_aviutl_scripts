# Z変位_Su

## 状況
MVP (v0.1.0) 作成完了 / 未検証 / 高速化計画済

## 対応内容
- 高速化計画: `.agents/plans/optimization.md` Z-1〜Z-3
- `Z変位_Su.anm2` 作成 (196行, UTF-8 BOMなし, CRLF)
- 輝度ベースの高さマップで画像メッシュをZ方向に変位させるアニメーション効果
  - 格子サイズに縮小したマップを tempbuffer に生成 → `getpixel` で輝度・アルファをサンプリング
  - `obj.drawpoly` による頂点Z付きポリゴン一括描画（回転・カメラ制御に追従する真の3Dレリーフ）
- 面モード3種 (`--select`)
  - 連続面: 頂点 (nx+1)×(ny+1) でサンプリングし隣接quadと頂点Zを共有 → 面が折れ曲がる見た目
  - ブロック: セル nx×ny、4頂点に同一Z → ピクセル積層の見た目
  - ブロック(側面あり): 高さ差0.5px超の隣接セル間に壁ポリゴン追加（カリング無効で描画）
- パラメータ: 強さ(±2000), 分割数X/Y(1〜192, 既定32, グループ化), 基準[%], マップぼかし, 裏面非表示, 透明部分スキップ
- 透明頂点は z=0 でスカート状に繋ぎ、全頂点透明のquadは描画スキップ
- 実績パターンのみ使用: `@立体化(g)` (drawpoly+z座標, culling), `引き伸ばし` (tmp縮小描画), `@DataEffect` (pixeloption/getpixel, cache運用)
- 構文チェック未実施（環境にLuaチェッカー無し）
