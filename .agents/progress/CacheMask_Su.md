## 対応内容

- v1.4.0: マスク配置モード追加 + premultiplied alpha 対応
  - `配置モード` select — サイズ合わせる(0) / 元サイズ配置(1)
  - `マスク位置X/Y` track — 元サイズ配置時のオフセット（ピクセル、ソース中心基準）
  - `マスク拡大率[%]` track — 元サイズ配置時のスケール（100%=等倍）
  - シェーダー内でUV変換、範囲外はマスク透明として処理
  - 出力を premultiplied alpha に修正（`src.rgb * l`）— 明るいソースが白く飛ぶ問題を解決
  - 輝度計算から `* m.a` 削除（premultiplied で二重乗算になっていた）
  - `配置モード` 初期値を 元サイズ配置(1) に変更
  - `マスクぼかし` track 追加 — マスクにぼかしをかけてエッジを柔らかくする
